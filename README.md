# PRECISIONHEART: HEART DISEASE PREDICTION

## Overview
PrecisionHeart is a hybrid ensemble framework designed for robust heart disease prediction. It combines Machine Learning and Deep Learning techniques to provide highly accurate and interpretable clinical results.

## Key Features
- **Hybrid Model:** Combines Gradient Boosting (ML) and Attention-augmented MLP (Deep Learning).
- **High Accuracy:** Achieved **93% accuracy** on benchmark datasets.
- **Explainable AI:** Uses **SHAP explanations** to help doctors understand the reasons behind each prediction.

## Technologies Used
- Python
- Machine Learning (Gradient Boosting)
- Deep Learning (Attention Mechanisms)
- SHAP (Interpretability Tool)
